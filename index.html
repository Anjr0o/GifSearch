About

This site was created to demonstrate proficiency in the use of 3rd Party APIs. API, which stands for "application programming interface", is a computing interface which simply defines the interactions between multiple software intermediaries such as the kinds of calls or requests that can be made, and how to make them. For example, I used GIPHY's API to send your search input and return a number of GIFs back. You'll also notice if you refresh the page, there will be a brand new background courtesy of Unsplash's API.
